# ⌨️ Hyprland Keybinds — Shadow Dragon

> `Super` = Windows key

---

## App Launcher

| Shortcut | Action |
|---|---|
| `Super + Enter` | Terminal (Kitty) |
| `Super + Shift + Enter` | App launcher (Rofi) |
| `Super + E` | File manager (Thunar) |
| `Super + V` | Clipboard history |

---

## Window Management

| Shortcut | Action |
|---|---|
| `Super + Q` | Tutup window |
| `Super + F` | Fullscreen |
| `Super + Shift + F` | Toggle float |
| `Super + P` | Pseudo tile |
| `Super + J` | Toggle split |

---

## Focus Navigation

| Shortcut | Action |
|---|---|
| `Super + H` | Fokus kiri |
| `Super + L` | Fokus kanan |
| `Super + K` | Fokus atas |
| `Super + J` | Fokus bawah |
| `Super + ← ↑ ↓ →` | Fokus (arrow key) |

---

## Pindah Window

| Shortcut | Action |
|---|---|
| `Super + Shift + H/L/K/J` | Geser window |

---

## Resize Window

| Shortcut | Action |
|---|---|
| `Super + Alt + H/L/K/J` | Resize window |

---

## Workspace

| Shortcut | Action |
|---|---|
| `Super + 1-0` | Pindah ke workspace 1–10 |
| `Super + Shift + 1-0` | Pindah window ke workspace |
| `Super + S` | Toggle scratchpad |
| `Super + Shift + S` | Pindah window ke scratchpad |
| `Super + Scroll` | Scroll antar workspace |

---

## Mouse

| Shortcut | Action |
|---|---|
| `Super + LClick` | Geser window |
| `Super + RClick` | Resize window |

---

## Screenshot

| Shortcut | Action |
|---|---|
| `Print` | Screenshot area → edit (Swappy) |
| `Super + Print` | Screenshot fullscreen → edit |
| `Super + Shift + P` | Screenshot area → clipboard |

---

## Power / Session

| Shortcut | Action |
|---|---|
| `Super + Escape` | Lock screen |
| `Super + Shift + Q` | Power menu (Wlogout) |

---

## Media Keys

| Shortcut | Action |
|---|---|
| `Vol+ / Vol-` | Volume naik / turun |
| `Mute` | Mute speaker |
| `MicMute` | Mute mikrofon |
| `Bright+ / Bright-` | Brightness naik / turun |
| `Play/Pause` | Play / pause |
| `Prev / Next` | Lagu sebelumnya / berikutnya |
